# 🎓 한장 — 한국외대 장학금, 한 번에

한국 대학생이 흩어져 있는 **교내·교외 장학금을 자기 조건으로 자동 매칭**하고,
토스처럼 **원클릭으로 신청**할 수 있게 하는 서비스의 MVP입니다.
1차 지원 대학은 **한국외국어대학교**이며, 점차 확대할 계획입니다.

## 주요 기능 (v0.1)

- **1분 온보딩** — 단과대학·학년·직전학기 평점·학자금 지원구간·특별자격(수급자/다자녀/보훈 등) 입력
- **맞춤 매칭 엔진** — 15종 장학금(교내 7 · 교외 8)의 자격요건을 규칙 기반으로 진단
  - `신청 가능` / `지원 가능·선발 심사` / `정보 입력 필요` / `요건 미충족` 4단계 판정
  - 요건별 충족/미충족 사유를 상세 화면에서 투명하게 표시
- **원클릭 신청** — 성적·재학증명서 자동 첨부 개념, 가능한 장학금 **N건 일괄 신청**
- **신청 현황 추적** — 신청완료 → 서류검토 → 심사 → 선정발표 진행 표시
- **마감 D-day 관리** — 홈에서 마감 임박 장학금 우선 노출

## 실행

빌드 과정이 없는 정적 웹앱입니다.

```bash
python3 -m http.server 8000
# http://localhost:8000 접속 (모바일 뷰 권장, max-width 480px)
```

## 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 화면 구조 (온보딩 · 홈 · 장학금 탐색 · 신청내역 · MY) |
| `style.css` | 토스 스타일 모바일 우선 UI |
| `data.js` | 장학금 데이터셋 + 자격요건 규칙 (시연용 예시) |
| `app.js` | 매칭 엔진 · 화면 렌더링 · 신청 플로우 · localStorage 상태 |
