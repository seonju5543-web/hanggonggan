# 🎓 한장 — 대학 장학금, 한 번에

한국 대학생이 흩어져 있는 **교내·교외 장학금을 자기 조건으로 자동 매칭**하고,
토스처럼 **원클릭으로 신청**할 수 있게 하는 서비스의 MVP입니다.
**전국 대학**을 지원하며, 설치형 PWA로 동작합니다.

## 주요 기능 (v0.2)

- **전국 대학 지원** — 학교 검색 선택(주요 55개 대학 자동완성 + 직접 입력), 전공 계열 기반 매칭
- **1분 온보딩** — 학교·계열·학년·직전학기 평점·학자금 지원구간·특별자격(수급자/다자녀/보훈 등) 입력
- **맞춤 매칭 엔진** — 교외(국가장학금 Ⅰ·Ⅱ유형, 국가근로, 인문100년, 대통령과학, 재단 등 9종) + 재학 대학 기준 교내 장학금(6~7종) 자격요건을 규칙 기반으로 진단
  - `신청 가능` / `지원 가능·선발 심사` / `정보 입력 필요` / `요건 미충족` 4단계 판정
  - 요건별 충족/미충족 사유를 상세 화면에서 투명하게 표시
- **원클릭 신청** — 증명서 자동 첨부 개념, 가능한 장학금 **N건 일괄 신청**
- **신청 현황 추적** — 신청완료 → 서류검토 → 심사 → 선정발표 진행 표시
- **PWA** — 홈 화면에 설치 가능, 오프라인 캐시 지원 (앱스토어 진출의 기반)

## 실행 / 배포

빌드 과정이 없는 정적 웹앱입니다.

```bash
# 로컬 실행
python3 -m http.server 8000
# http://localhost:8000 접속 (모바일 뷰 권장, max-width 480px)
```

**GitHub Pages 배포**: `.github/workflows/deploy-pages.yml`이 push마다 자동 배포합니다.
저장소 Settings → Pages → Source를 **GitHub Actions**로 한 번만 설정하면
`https://<계정>.github.io/hanggonggan/` 에서 접속할 수 있습니다.

**앱스토어 진출 경로**:
1. (현재) PWA — 브라우저에서 "홈 화면에 추가"로 설치
2. Android: [Bubblewrap](https://github.com/GoogleChromeLabs/bubblewrap)으로 TWA 패키징 → Google Play 등록 (개발자 계정 $25 1회)
3. iOS: [Capacitor](https://capacitorjs.com)로 래핑 → App Store 등록 (개발자 계정 $99/년, macOS + Xcode 필요)

## 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 화면 구조 (온보딩 · 홈 · 장학금 탐색 · 신청내역 · MY) |
| `style.css` | 프리미엄 모바일 우선 UI (Pretendard, 딥 네이비 팔레트) |
| `data.js` | 교외 장학금 데이터 + 교내 장학금 템플릿 + 자격요건 규칙 (시연용) |
| `app.js` | 매칭 엔진 · 화면 렌더링 · 신청 플로우 · localStorage 상태 |
| `manifest.json` / `sw.js` / `icons/` | PWA (설치·오프라인) |
| `BUSINESS.md` | 수익화 전략 |
