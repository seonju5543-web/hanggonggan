# 완전 자동 접수 시스템 (5-1의 2번)

## 구조
사용자 [자동 접수] 탭 → 앱이 신청 내용+서류를 이 서버로 전송 → 서버가 접수처 이메일로 대신 발송 → 사용자에게 발송 확인.

## 현재 상태
- 서버 코드(`mail-worker.js`) 완성 — 배포 대기
- 앱은 `HANDAEJANG_CONFIG.mailEndpoint`가 설정되는 순간 자동 접수 버튼이 활성화되도록 준비됨
- **개발자 액션 필요 (10분)**: resend.com API 키 + Cloudflare Worker 배포 (절차는 mail-worker.js 상단 주석)
- 그 전까지는 [접수 메일 열기] 버튼(받는 사람·제목·본문 자동 완성)이 대체 경로로 동작
